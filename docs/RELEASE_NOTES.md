# 发布说明 / Release Notes

## v0.2.1

> 相对 v0.2.0 的更新。本版重点：**背景去除后处理**、批量导出前端、帧多选与播放、HTML5 视频预览、自定义平滑滚轮。

### ✨ 新功能

#### 1. 背景去除（抠像）后处理 — 核心新功能
视频处理完成后进入独立的背景去除面板，支持实时预览后再应用到全部帧。

- **自动取色**：默认从图像四条边采样取中位数作为背景色，能正确处理 `#fefefe` 这类轻微偏色 / 压缩偏色背景，无需手动取色。
- **手动取色**：可关闭自动取色，用取色器或填写十六进制色值。
- **软 Alpha 边缘**：用两段阈值 `[t0, t1]` + smoothstep 替代旧的硬布尔遮罩，过渡区半透明，边缘抗锯齿而非锯齿状。
- **边界连通识别主体**：从图像四边 flood-fill，只抠与边界连通的背景区，主体内部与背景同色的像素因不连通而保留 —— 不再在角色身上打洞。
- **去抖动 / 去碎点**：3×3 中值滤波压掉压缩抖动（`fefefe/fdfdfd` 跳变），小连通域面积过滤清除残留碎点，比 SLIC 超像素更轻、无新增依赖。
- **已透明帧保护**：检测到输入帧已带有效 Alpha（>2% 透明像素）时直接跳过，避免在已抠像帧上二次误吃主体暗部。
- **可选羽化**：`feather` 滑块对 Alpha 做额外高斯软化。
- **导出保留透明**：导出层改用 `IMREAD_UNCHANGED` 读取，背景去除后的 BGRA 帧在 PNG 序列 / GIF / 精灵图 / 单张导出时保留透明通道。

#### 2. 批量导出前端
- 导出对话框支持四种格式：PNG 序列 + `manifest.json`、动画 GIF、4×4 精灵图、单张 PNG。
- 帧选择器（all / range / indices / current）、`every_n_frames`、`target_fps` 时间轴重采样、`max_frames` 上限。
- 索引选择器重设计：预览分栏布局 + 重复路径自动改名（`png_sequence` 文件夹导出跳过自动改名以保留原始命名）。
- 导出 / 下载入口移至侧栏信息面板，置于帧序列列表下方。

#### 3. 帧多选与播放
- 视觉化多选帧拾取器；shift-click 范围多选。
- 选中区间内播放循环；播放/拖动/点击自动将当前帧滚入视口。
- 缩略图滚动与用户交互解耦，支持独立的滚轮平移。

#### 4. HTML5 视频预览
上传视频后直接在应用内预览原视频，无需外部播放器。

#### 5. 自定义平滑滚轮
- 帧列表与左侧设置面板均改为带惯性 / 动量的自定义滚轮滚动。
- 当前帧对齐使用缓动（先快后慢）动画，并与惯性取消器联动。
- 修复设置面板滚轮监听器在 ready 过渡后才绑定的问题。

### 🛠 改进与修复

- **Tauri CWD 路径冲突解析**：解决 Tauri 外壳与后端工作目录冲突。
- **Finder 弹窗修复**：`open_path_in_finder` 将相对路径解析为绝对路径，修复 macOS 上 Finder 反复弹窗。
- **导出默认路径**：默认输出到 `output/` 目录，避免弄脏 git index。
- 导出尺寸 / 适配模式 / 精灵图填充策略等导出参数对齐后端 API。

### 🧹 工程

- 停止追踪并忽略 Python `__pycache__` 缓存与测试导出产物。

### 📦 升级说明

桌面端用户：直接下载新版 `.dmg`（macOS）/ `.exe`（Windows）安装即可。本版后端 sidecar 重新打包，包含新的背景去除接口；旧版应用无法使用背景去除功能。
